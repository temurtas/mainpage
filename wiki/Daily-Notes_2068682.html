12 Jul 2017 

Pomotodo Daily Report

20170712-08

:Notes:

  •  Tried to find a way to drive servo from Raspberry and failed. Decided to use Arduino as a motor controller instead. I will connect digital outputs of Raspberry to the digital pins of Arduino. With the help of digitalRead function, I will control servo. 
  • Initial body of code's constructed.
  • Got used to using Linux terminal and using git from it. Now I can create, eddit, execute and stop files or programs without leaving terminal screen.
    • touch file_name -→ creates file
    • nano file_name -→ opens in editor(nano), allows you to edit
    • Ctrl+z -→ stops programs executing
    • git add file_name --→ adds file to git repo
    • git commit file_name -→ commits
    • git push origin master -→ sends all commits in master branch to server
